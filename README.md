<h3 id="URPC2017">URPC2017</h3>

- 17655张训练，985张测试，分辨率720x405.
- 所有图像均取自6个视频，间隔为10帧
- 所有的视频都是在人工模拟的环境中拍摄，因此同一个视频中的图片几乎完全一样

![image-20240204194025536](figs/image-20240204194025536.png)
